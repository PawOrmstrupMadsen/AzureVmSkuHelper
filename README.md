# AzureVmSkuHelper

Run like this:
```
.\AzureVmSkuHelper.exe --SubscriptionId "<insert Subscription ID>"
```

Options are:
```
  --SubscriptionId      Required.
  --Location            (Default: norwayeast)
  --CoreCount           (Default: 4)
  --Memory              (Default: 8)
  --PreferredFamily     (Default: B)
  --PreferredCpuType    (Default: AMD)
  --help                Display this help screen.
  --version             Display version information.
```
